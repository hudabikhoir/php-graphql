# PHP-GRAPHQL
Clean and simple single-file example of main GraphQL concepts originally proposed

### Run locally
```
php -S localhost:8080 
```
### Try Running on Browser
```
http://localhost:8080/pegawai.php
```

### Try query
open pegawai.php
```
$processor->processPayload('{ pegawai(id:1){id,nama} }');
```
